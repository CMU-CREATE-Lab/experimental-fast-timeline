getMinMaxValuesWithinTimeRange(minTimeSecs, maxTimeSecs) → {MinMaxValue}

Returns the minimum and maximum values for points within the given time range. Returns null if there's no data within the time range, no data loaded within the time range, or if the minTimeSecs is greater than the maxTimeSecs. Since this method does not proactively load data, it limits the requested time range to be within the current date rage.
Parameters:
